# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through [GitHub Security Advisories](https://github.com/dvashim/typescript-config/security/advisories/new).

Please do **not** open a public GitHub issue for security vulnerabilities.

You should expect an initial response within 72 hours. Once the issue is confirmed, a fix will be released as soon as possible.
